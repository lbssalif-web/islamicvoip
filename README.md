🌟 Dua Stars: ดุอาอ์ดวงดาราน้อยสำหรับคุณแม่และหนูน้อยDua Stars คือเว็บแอปพลิเคชันรูปแบบใหม่ที่ออกแบบมาเพื่อช่วยเหลือคุณพ่อคุณแม่มุสลิมในการอบรมสั่งสอนลูกรักให้คุ้นเคยและท่องจำดุอาอ์ในชีวิตประจำวันตามแบบฉบับสุนนะฮ์อันดีงามของท่านนบีมูฮัมหมัด (ซ.ล.) ได้อย่างมีความสุข สนุกสนาน และมีเป้าหมาย👉 [คลิกเพื่อเข้าชมและใช้งานเว็บแอปพลิเคชันฟรี!] (ใส่ URL ของ GitHub Pages ของคุณที่นี่หลังจากเปิดใช้งาน)✨ ฟีเจอร์หลักของแอปพลิเคชัน (Key Features)ดุอาอ์ที่มีหลักฐานถูกต้อง 100% (Authentic Sunnah):รวบรวมดุอาอ์ที่จำเป็นในชีวิตประจำวันจำนวน 20 บททุกบทได้รับการตรวจสอบหลักฐานอ้างอิงและผู้รายงานจากตารางหะดีษศอฮีห์และหะซัน (เช่น อัล-บุคอรีย์, มุสลิม, อบู ดาวุด, อัต-ติรมิซีย์) ปราศจากดุอาอ์อุปโลกน์ระบบแบ่งหมวดหมู่แสนสวยงาม (Cute Structured Tabs):จัดสัดส่วนดุอาอ์ตามหมวดหมู่เพื่อการเรียนรู้อย่างเป็นขั้นตอน:🌅 ยามเช้าและกลางคืน🍽️ กินดื่มและร่างกาย💖 ครอบครัวและมารยาท📚 ความรู้และการคุ้มครองระบบสังเคราะห์เสียงอัจฉริยะ (Dual-Engine TTS with Gemini Voice):ฟังเสียงอาหรับสำเนียงแท้: มีปุ่มกดฟังเสียงอ่านภาษาอาหรับที่ถูกต้องตามอักขระวิธีฟังเสียงเด็กน้อยแปลความหมาย: นวัตกรรมจำลองน้ำเสียงเด็กไทยสุดน่ารักและอ่อนโยนผ่านระบบ Gemini AI เพื่อให้เด็ก ๆ เข้าใจความหมายได้ง่ายและอบอุ่นใจกระตุ้นการเรียนรู้ด้วยเกม (Gamification for Kids):ระบบให้ดาวสะสมความดีเมื่อท่องจำดุอาอ์ได้สำเร็จครั้งละ 5 ดาว ⭐แสดงพลุกระดาษฉลองความสำเร็จ (Confetti) พร้อมเสียงเอฟเฟกต์แสนน่ารักเมื่อสะสมดาวหรือเมื่อเด็ก ๆ หัดท่องดุอาอ์ครบ 3 ครั้งตามหลักสุนนะฮ์คู่หูมาสคอตเลือกได้ (Choose Your Companion):เด็ก ๆ สามารถเปลี่ยนคู่หูคอยให้กำลังใจได้ (⭐️ ดาราน้อย, 🌙 เสี้ยวจันทร์, ☀️ ตะวันยิ้ม, ☁️ เมฆปุย)ระบบซิงก์คลาวด์แบบเรียลไทม์ (Real-time Cloud Sync):บันทึกความก้าวหน้าและคะแนนดาวข้ามเครื่องข้ามอุปกรณ์ด้วยรหัสผ่านความปลอดภัย (Sync Code) โดยไม่มีข้อมูลสูญหาย🛠️ เทคโนโลยีที่ใช้พัฒนา (Tech Stack)HTML5 & Vanilla JavaScript: สำหรับควบคุมระบบและประมวลผลภายในทั้งหมดTailwind CSS: ในการออกแบบส่วนติดต่อผู้ใช้ (UI) ที่มีความนุ่มนวล เป็นมิตร และตอบสนองต่อทุกขนาดหน้าจอ (Fully Responsive)Web Audio API: สำหรับสังเคราะห์และกำเนิดเสียงเอฟเฟกต์โดยไม่มีการดึงไฟล์จากภายนอกGemini API & Web Speech Synthesis: ระบบแปลงข้อความเป็นเสียงพูด (Text-to-Speech)Firebase (Firestore & Auth): สำหรับรันระบบระบบออนไลน์คลาวด์ซิงก์เรียลไทม์📂 โครงสร้างโปรเจกต์ (Project Structure)เนื่องจากโปรเจกต์นี้ได้รับการพัฒนาให้เป็นแอปพลิเคชันแบบหน้าเดียวที่ลื่นไหล (Single-Page Application) โครงสร้างไฟล์จึงเรียบง่ายและง่ายต่อการอัปโหลดขึ้นเซิร์ฟเวอร์:dua-stars/
